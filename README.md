# Kode Wilayah Kota Malang SQL Database
Database Kode Wilayah Kota Malang bagi yang membutuhkan untuk form select pada aplikasi.

1. Kecamatan Blimbing (11 Kelurahan)
2. Kecamatan Klojen (11 Kelurahan)
3. Kecamatan Kedungkandang (12 Kelurahan)
4. Kecamatan Sukun (11 Kelurahan)
5. Kecamatan Lowokwaru (12 Kelurahan)

- Tested in : `PHP 8.3.26`  `MySQL 8.4.3`
- Reference : [kodewilayah.id](https://kodewilayah.id/35.73), [id.wikipedia.org](https://id.wikipedia.org/wiki/Daftar_kecamatan_dan_kelurahan_di_Kota_Malang)
